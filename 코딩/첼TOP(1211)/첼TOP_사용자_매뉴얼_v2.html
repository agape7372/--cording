1. 프로그램 소개

1.1 첼TOP이란?

첼TOP(Chello Treatment Optimizer & Planner)은 재활의학과 치료 시간표 관리를 위한 웹 기반 애플리케이션입니다.

왜 필요한가요?
재활의학과에서는 매일 수십 명의 환자에게 다양한 치료를 제공합니다. 복잡한 일정을 수기로 관리하면 충돌, 누락, 비효율적인 동선 등의 문제가 발생합니다. 첼TOP은 이러한 문제를 해결합니다.

시스템 요구사항

항목권장 사양
브라우저Chrome, Edge (최신 버전)
운영체제Windows, macOS, Android, iOS
인터넷최초 로딩 시에만 필요

1.2 주요 기능 요약

첼TOP은 세 가지 핵심 기능을 제공합니다.

📊 리뷰 도우미

시간표를 분석하여 치료사/환자 충돌을 감지하고 통계를 제공합니다.

📅 아침 시간표

환자별 맞춤 시간표를 자동 생성하고 HTML로 저장합니다.

📖 신규 가이드

신규 직원을 위한 재활센터 업무 매뉴얼입니다.

2. 리뷰 도우미

2.1 기능 설명

치료 시간표를 분석하여 문제점을 사전에 파악하는 기능입니다.

기능설명
시간표 분석엑셀 파일에서 치료사별 일정 추출
충돌 감지동일 시간대에 같은 치료사/환자 중복 배정 경고
치료사 통계PT/OT별 치료 건수 집계
시간대별 현황각 시간대별 가용/바쁜 치료사 표시

2.2 사용 방법

1시작 화면에서 '리뷰 도우미' 카드를 클릭합니다.
2시간표 엑셀 파일(.xlsx)을 업로드합니다.
3분석 결과를 확인합니다.
💡 팁
충돌이 감지되면 해당 시간대와 관련 치료사/환자가 빨간색으로 표시됩니다.

3. 아침 시간표

3.1 기능 설명

환자별 맞춤 시간표를 자동으로 생성하고 저장하는 핵심 기능입니다.

기능설명
환자 검색이름 입력 시 모든 시간표 파일에서 일정 추출
시간표 생성08:30~17:00 시간대별 치료 항목 자동 배치
처방 해석RM, W, P 등 처방 코드를 한글 치료명으로 변환
층 이동 표시2층/지하1층 이동 필요 치료에 ↓↑ 표시
HTML 저장브라우저에서 편집 가능한 HTML로 저장
엑셀 합치기여러 시간표 파일을 하나로 병합
자동완성기존 파일에서 차트번호/호실 자동 인식

3.2 사용 방법

1시작 화면에서 '아침 시간표' 카드를 클릭합니다.
2층별 시간표 엑셀 파일(최대 3개)을 업로드합니다.
3'환자 목록 입력' 칸에 환자 정보를 입력합니다.
4'🚀 시간표 생성하기' 버튼을 클릭합니다.
5생성된 시간표를 확인하고 '💾 HTML 저장' 버튼으로 저장합니다.

환자 정보 입력 방법

입력 형식: 이름 차트번호 호실 (공백으로 구분)

홍길동 111111 501
김영희 222222 302
박철수
차트번호나 호실 없이 이름만 입력해도 됩니다. 폴더 연결 시 기존 파일에서 자동완성됩니다.

3.3 저장 폴더 연결 (PC 전용)

저장 폴더를 미리 연결하면 매번 위치를 선택하지 않아도 됩니다.

연결 상태저장 위치
연결 O지정한 폴더에 자동 저장
연결 X기본 다운로드 폴더에 저장

폴더 연결 방법

1'📂 저장 폴더 미리 연결하기' 버튼을 클릭합니다.
2파일을 저장할 폴더를 선택합니다.
3"✅ 연결됨: [폴더명]" 메시지가 표시되면 완료!
⚠️ 주의
• 폴더 연결은 PC(Chrome/Edge)에서만 지원됩니다.
• 브라우저 탭을 닫으면 연결이 해제됩니다.

3.4 자동완성 기능

폴더 연결 시 기존 파일에서 차트번호와 호실을 자동으로 가져옵니다.

기존 파일입력결과 파일명
(파일 없음)홍길동홍길동-______-____
홍길동-111111홍길동홍길동-111111-____
홍길동-501홍길동홍길동-______-501
홍길동-111111-501홍길동홍길동-111111-501 ✨
💡 팁
이름만 입력해도 기존 파일에서 차트번호, 호실을 가져와요!

3.5 엑셀 합치기

여러 개의 시간표 엑셀 파일을 하나의 파일로 합칠 수 있습니다.

12개 이상의 엑셀 파일을 업로드합니다.
2헤더에 📑 합치기 버튼이 나타납니다.
3버튼을 클릭하면 '합본_MMDD.xlsx' 파일이 다운로드됩니다.
합쳐진 파일은 각 원본 파일의 시트명을 유지하며, 서식/스타일/숨겨진 행도 모두 보존됩니다.

3.6 HTML 편집 및 재저장

저장된 HTML 파일은 브라우저에서 직접 열어 편집하고 다시 저장할 수 있습니다.

1저장된 HTML 파일을 더블클릭하여 브라우저에서 엽니다.
2시간표 셀을 클릭하면 편집 모드로 진입합니다.
3내용을 수정합니다.
4우측 상단의 💾 저장 버튼을 클릭합니다.
5🖨️ 인쇄 버튼으로 바로 인쇄할 수 있습니다.
💡 팁
첫 저장 시 위치 선택 후, 이후 저장은 같은 위치에 자동으로 덮어씁니다.

4. 신규 가이드

4.1 기능 설명

신규 직원을 위한 재활센터 업무 매뉴얼입니다. 치료 관련 정보와 업무 프로세스를 한눈에 확인할 수 있습니다.

기능설명
검색 기능키워드로 필요한 정보를 빠르게 검색
글자 크기 조절가독성에 맞게 글자 크기 조절 가능
카테고리 분류치료 종류별, 업무별로 분류된 정보
차팅 내용 복사자주 쓰는 차팅 문구를 클릭 한 번으로 복사

디자인 특징

요소설명
2026 팬톤 올해의 컬러눈의 피로를 줄이는 파스텔 톤 색상 테마 적용
듀오톤 아이콘카테고리별 자체 디자인 아이콘으로 직관적 구분

4.2 사용 방법

1시작 화면에서 '신규 가이드' 카드를 클릭합니다.
2상단 검색창에서 원하는 키워드를 검색합니다.
3카테고리를 클릭하여 세부 내용을 확인합니다.

차팅 내용 복사 기능

자주 사용하는 차팅 문구를 클릭 한 번으로 클립보드에 복사할 수 있습니다.

1원하는 차팅 문구 영역을 찾습니다.
2문구를 클릭하면 자동으로 복사됩니다.
3EMR 등 원하는 곳에 붙여넣기(Ctrl+V) 합니다.
💡 팁
복사 완료 시 화면에 "복사됨!" 알림이 표시됩니다.
신규 가이드는 오프라인에서도 사용 가능합니다. 한 번 로딩 후에는 인터넷 연결 없이도 볼 수 있습니다.

6. 자주 묻는 질문 (FAQ)

Q. 모바일에서도 사용할 수 있나요?

네, 기본 기능은 모바일에서도 사용 가능합니다. 단, '폴더 연결' 기능은 PC(Chrome/Edge)에서만 지원됩니다.

Q. 인터넷 연결이 필요한가요?

최초 로딩 시에만 필요합니다. 이후에는 오프라인에서도 사용 가능합니다.

Q. 엑셀 파일이 읽히지 않아요.

.xlsx 또는 .xls 형식인지 확인해주세요. 비밀번호가 걸린 파일은 지원되지 않습니다.

Q. 저장한 HTML 파일을 다른 PC에서 열 수 있나요?

네, HTML 파일은 어떤 PC의 브라우저에서든 열 수 있습니다. 편집과 재저장도 가능합니다.

Q. 합치기한 엑셀 파일의 서식이 깨져요.

합치기 기능은 서식을 최대한 보존하지만, 일부 복잡한 서식은 완벽히 복사되지 않을 수 있습니다.

Q. 처방 코드가 인식되지 않아요.

지원되지 않는 처방 코드일 수 있습니다. 로그 창에서 '미인식 데이터'를 확인해주세요.
