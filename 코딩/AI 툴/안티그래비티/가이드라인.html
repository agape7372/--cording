첼 TOP 어플리케이션 가이드

1. 기대효과 (Expected Effects)

이 어플리케이션은 병원 재활치료 스케줄 관리와 환자 안내 업무를 효율화하기 위해 설계되었습니다.

  • 업무 자동화 및 시간 단축: 수작업으로 진행하던 엑셀 스케줄 분석과 환자별 시간표 작성을 자동화하여 치료사와 행정 직원의 업무 시간을 획기적으로 단축합니다.
  • 휴먼 에러 방지: 스케줄 중복(Conflict), 처방 누락, 배정 오류 등을 시스템이 자동으로 감지하여 시각적으로 알려줌으로써 실수를 예방합니다.
  • 환자 서비스 품질 향상: 환자 개개인에게 깔끔하고 정확한 맞춤형 치료 시간표(PDF/인쇄물)를 신속하게 제공하여 병원의 전문성을 보여주고 환자 만족도를 높입니다.
  • 통합 워크플로우: 스케줄 검토(리뷰)와 출력물 생성(아침 시간표) 업무를 하나의 인터페이스에서 처리할 수 있어 업무 흐름이 끊기지 않습니다.
  • 데이터 보안성: 별도의 서버 전송 없이 사용자의 브라우저 내에서 모든 데이터가 처리되므로, 환자 개인정보 유출 걱정 없이 안전하게 사용할 수 있습니다.

2. 기능적 강점 및 효율성

기능적 강점 (Functional Strengths)

  • 듀얼 모드 시스템: '리뷰 도우미'와 '아침 시간표' 기능을 탭 하나로 전환하며 유기적으로 사용할 수 있습니다.
  • 직관적인 시각화: 텍스트로 된 엑셀 데이터를 타임라인 형태의 시각적 도표로 변환하여, 치료 흐름을 한눈에 파악할 수 있습니다.
  • 지능형 충돌 감지: 치료사 및 환자의 스케줄 중복을 실시간으로 분석하고, 대체 가능한 치료사를 추천하는 스마트한 기능을 제공합니다.
  • 강력한 호환성: 별도의 프로그램 설치 없이 브라우저(Chrome, Edge)만 있으면 어디서든 실행 가능하며, 다양한 엑셀 서식을 유연하게 처리합니다.

효율성 (Efficiency)

  • 작업 속도 향상: 수십 명의 환자 스케줄을 일일이 대조하던 작업을 수 초 내에 완료하여 업무 효율을 극대화합니다.
  • 원클릭 출력: 복잡한 설정 없이 환자 목록만 입력하면, 즉시 인쇄 가능한 형태의 PDF를 생성합니다.
  • 자원 절약: 종이로 출력하여 검토하던 과정을 화면상에서 해결하여 불필요한 종이 낭비를 줄입니다.
  • 유지보수 용이성: 단일 HTML 파일로 구성되어 배포와 관리가 매우 간편합니다.

기능별 상세 강점 (Feature Highlights)

Review 리뷰 도우미 (Review Helper)

  • 강점: 전체 치료사의 스케줄을 한 화면에 통합하여, 빈 시간(Void)과 중복(Conflict)을 직관적으로 식별할 수 있습니다.
  • 효율: 치료사별 처방 건수(PT/OT)를 자동 집계하여, 업무 불균형을 즉시 파악하고 조율할 수 있습니다.

Morning 아침 시간표 (Morning Schedule)

  • 강점: 환자별 맞춤형 시간표를 A4 용지 규격에 맞춰 자동으로 레이아웃을 잡아주어, 별도의 편집 과정이 필요 없습니다.
  • 효율: 다수의 환자 명단을 일괄 처리하여, 수십 장의 시간표를 단 몇 번의 클릭만으로 생성하고 저장할 수 있습니다.

3. 사용 가이드라인 (Guidelines)

A. 공통 사항

  • 권장 환경: 최신 기능을 지원하는 Chrome(크롬) 또는 Edge(엣지) 브라우저 사용을 권장합니다.
  • 데이터 처리: 파일 업로드 시 서버로 전송되지 않고 로컬에서만 처리됩니다. 새로고침 시 작업 내용이 초기화될 수 있으니 주의하세요.

B. 리뷰 도우미 (Review Helper)

목적: 전체 치료 스케줄의 정합성 검토 및 환자별 처방 이행 현황 분석

  1. 파일 업로드:
    • 메인 화면 또는 상단의 '파일 선택' 영역에 스케줄 엑셀 파일(.xlsx, .csv)을 드래그하거나 클릭하여 업로드합니다.
  2. 통합 시간표 확인 (Visual Tab):
    • 업로드 후 자동으로 스케줄이 시각화됩니다.
    • 중복 감지: 같은 시간에 치료사나 환자가 중복 배정된 경우 붉은색으로 표시됩니다.
    • 해결 제안: 중복된 항목을 클릭하면 해당 시간에 비어있는 치료사(PT/OT) 목록을 추천받을 수 있습니다.
  3. 처방 분석 (Stats Tab):
    • 환자별로 배정된 치료 횟수와 처방 코드를 분석합니다.
    • 설정된 기준(Target)에 따라 정상(초록), 부족(빨강), 초과(노랑) 상태를 한눈에 파악할 수 있습니다.

C. 아침 시간표 (Morning Schedule)

목적: 환자 개인별 치료 시간표 생성 및 출력 (회진/안내용)

  1. 스케줄 파일 로드:
    • '메인 파일' 및 필요 시 '추가 파일' 슬롯에 스케줄 엑셀 파일을 업로드합니다.
  2. 환자 목록 입력:
    • 시간표를 출력할 환자의 이름과 호실을 입력합니다.
    • 입력 형식: 이름 호실 (예: 홍길동 501)
    • 여러 명일 경우 줄바꿈(Enter)으로 구분하여 입력합니다.
  3. 시간표 생성 및 저장:
    • 시간표 생성하기: 버튼을 누르면 하단에 미리보기가 생성됩니다.
    • 저장 폴더 연결: '저장 폴더 미리 연결하기'를 통해 PC의 특정 폴더를 선택하면, PDF 저장 시 해당 폴더에 즉시 저장됩니다.
    • PDF 자동 저장 / 전체 인쇄: 생성된 시간표를 파일로 저장하거나 종이로 출력합니다.

D. 주의사항

  • 엑셀 서식: 어플리케이션이 인식할 수 있는 표준 스케줄 양식을 준수해야 정확한 분석이 가능합니다. (치료사명, 시간대 형식 등)
  • 초기화: 작업 중 오류가 발생하거나 새로운 작업을 시작할 때는 우측 상단의 '리셋' 버튼을 활용하세요.
